# FastAPI Expense API

A beginner FastAPI CRUD API for tracking expenses.

## Tech Used
- Python
- FastAPI
- Pydantic
- Uvicorn
- Git/GitHub

## Features
- Create expense
- Get all expenses
- Get expense by ID
- Update expense
- Delete expense

## Endpoints
GET /
GET /expenses
POST /expenses
GET /expenses/{expense_id}
PUT /expenses/{expense_id}
DELETE /expenses/{expense_id}
